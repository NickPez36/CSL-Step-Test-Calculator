## Maintained application

Use the **React + Vite app** in [`react-app/`](react-app/). It is the current product: full **Athlete & Session Details** including **Temperature** and **Wind speed**, plus the structured PDF pipeline.

```bash
cd react-app
npm install
npm run dev
```
